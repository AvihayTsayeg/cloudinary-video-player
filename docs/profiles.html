Cloudinary Video Player

Profiles

Profiles are still under development and it's not safe for use on production.
For testing purpose you can use Profiles Feature according to the instructions below:

1. Provide `__profileName` config property using one of default profiles (`loopingVideo`, `videoStream`, `stylish`)
2. You can use your own profile JSON, create folder on your cloud named `profiles` and upload json file - once you have it just provide publicId with extension of your uploaded file as value of `__profileName`

Example Code:



      <video
        id="player-default-profile"
        controls
        autoplay
        muted
        class="cld-video-player"
        width="500">
      </video>

    <video
        id="player-custom-profile"
        controls
        autoplay
        muted
        class="cld-video-player"
        width="500">
      </video>

      
      
        // default profile
        cloudinary.videoPlayer('player', {
          cloud_name: 'demo',
          __profileName: 'loopingVideo',
        }, function () {
          player.source('sea_turtle');
        });

        // custom profile
        cloudinary.videoPlayer('player', {
          cloud_name: 'dwaq5xqm4',
          __profileName: 'default_profile.json',
        }, function () {
          player.source('samples/cld-sample-video');
        });
